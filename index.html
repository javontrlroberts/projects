CSS Colors Quick Reference

This CSS Cheat Sheet serves as a concise reference for the various color values used to style web elements. It provides a quick overview of opacity, HSL, RGB, hexadecimal, and named color values, offering a valuable tool for easily recalling and implementing different coloring options for foreground and background elements, enhancing the visual appeal of websites.

Color Type Syntax Description Example
Named Color color:
background-color: 		Consists of standard, easy-to-remember color keywords span { background-color: saddlebrown; }
Hexidecimal #RRGGBB hex colors begin with (#) followed by 3 or 6 characters representing red, blue, and green .black { background: #000; }
RGB rgb(255 255 255) Colors expressed in Red-Green-Blue (RGB), typically sequenced between 0 and 255 #teal { color: rgb(0, 128, 128); }
HSL hsl(120, 60%, 70%) Colors expressed according to Hue, Saturation, and Lightness; w/the first num representing the degree of hue between 0 and 360; and the second and third num being percentages representing saturation and lightness between 0% and 100% footer { background-color: hsl(0 100% 50%); }
Opacity and Alpha (Transparency of a Color)
Color Type Syntax Description Example
<alpha-value> rgb(34 12 64 / 60%) or
hsla(34, 100%, 50%, 0.1)		 A fourth value representing the "Alpha" value that can be a number between 0 (fully transparent) and 1.0 (fully opaque) with decimal values in-between; or percentages between 0% (fully transparent) and 100% (fully opaque) .opaque .transparent { color: rgba(234, 45, 98, 0.33); background-color: hsla(90 80% 50% / 50%); }
